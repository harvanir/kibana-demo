# Kibana Demo

## Instructions

### 1. Initialize

```shell
./init
```

### 2. Build application

```shell
./build
```

### 3. Run application

```shell
./run
```

### 4. Open Kibana dashboard

Open from your browser: http://localhost:5601/

### 5. Stop application

```shell
./stop
```

### 6. Reset & Clean up application
```shell
./cleanup
```
